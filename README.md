# Lore

**Lore** is a suite of AI-powered tools for prompt engineering, workflow continuity, and verified document pipelines.

[Read the manifesto](./MANIFESTO.md)

## Design language

- **Color:** Tyrian purple (`#7B2FBE`) on black
- **Type:** Cinzel serif, white
- **Voice:** Direct. No hedging.

---

## Philosophy

The tools in this suite exist because working with AI systems at any volume surfaces the same friction points: prompts that break in ways you didn't anticipate, context that evaporates when a session ends, and documents that take hours to produce manually for every new role or audience. Each tool addresses one of those problems directly.
